A OpenAI lançou a Library (Biblioteca) no ChatGPT: um espaço seguro onde todos os seus arquivos ficam salvos automaticamente. Chega de perder documentos importantes no meio de conversas antigas!

O que é a Library do ChatGPT?

Sabe quando você manda uma planilha, um PDF ou uma imagem para o ChatGPT analisar e, semanas depois, precisa usar aquele mesmo arquivo novamente? Antes, você teria que procurar no histórico de conversas ou fazer o upload de novo. Agora, não mais.

A Library é uma seção dedicada dentro do ChatGPT onde todos os arquivos que você envia ou que a IA gera ficam salvos automaticamente. É como uma nuvem pessoal dentro do próprio ChatGPT.
